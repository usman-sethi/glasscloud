# GlassCloud - Features

GlassCloud is a modern, sleek, and feature-rich file management web application designed for seamless cloud storage experience. Below is a comprehensive list of all the features available in the application:

## 📁 File Management
- **File Explorer Interface:** Intuitive navigation with both Grid and List views.
- **Folder Navigation:** Support for nested folders with breadcrumb navigation for easy traversal.
- **File Upload:** Drag-and-drop support and click-to-select functionality for uploading files.
- **Create Folders:** Easily organize your files by creating new folders.
- **Rename Files & Folders:** Quickly rename your items with a simple context menu action.
- **Move to Trash:** Safely delete files by moving them to a dedicated Trash bin.
- **Restore from Trash:** Recover accidentally deleted files from the Trash.
- **Permanent Deletion:** Permanently remove files from the Trash to free up space.

## ⭐ Organization & Discovery
- **Favorites:** Mark important files and folders as favorites for quick access in a dedicated "Favorites" view.
- **Search:** Real-time search functionality to quickly find files by name.
- **Sorting:** Sort files and folders by Name, Date Modified, or Size (Ascending/Descending).
- **Recent Files:** Quick access to recently modified or uploaded files.

## 🖼️ Media Handling
- **Media Preview (Lightbox):** Built-in lightbox to preview images and watch videos directly within the app without downloading.
- **Cloudinary Integration:** 
  - Generates highly optimized thumbnail URLs for images and videos to save bandwidth and improve load times.
  - Adaptive video streaming for smooth playback.
- **File Type Icons:** Visual indicators for different file types (Images, Videos, Documents, Archives, Audio, etc.).

## ⚡ Performance & UX
- **Infinite Scrolling:** Automatically loads more files as you scroll down the list, ensuring smooth performance even with thousands of files.
- **Storage Usage Indicator:** Visual progress bar and breakdown of storage usage by file type (Images, Videos, Documents, Others) out of a 5 GB limit.
- **Context Menu:** Custom right-click context menu on files and folders for quick actions (Rename, Favorite, Move to Trash, Download).
- **Responsive Design:** Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices.
- **Dark Mode UI:** A sleek, modern, and eye-friendly dark theme with glassmorphism elements.
- **Toast Notifications:** Non-intrusive toast notifications for successful actions (e.g., "File uploaded", "Folder created") and errors.
- **Optimistic UI Updates:** Instant UI feedback when performing actions like renaming or deleting, before the server confirms the change, for a snappier feel.

## 🔒 Security & Data
- **User Authentication:** Secure access to personal files (tied to User ID).
- **Persistent Metadata:** Local storage persistence for file metadata like "Favorites" and "Trashed" status.
